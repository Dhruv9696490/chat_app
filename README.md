<h2>Project Overview</h2>
<p>
This application replicates core WhatsApp functionality including authentication, 
real-time messaging, CRUD operations on messages, theme switching, and offline data handling.
</p>

<p>
The primary focus of this project was implementing production-level architecture 
rather than only replicating UI.
</p>

<hr>

<h2>Tech Stack</h2>

<ul>
<li>Flutter (Web & Mobile)</li>
<li>Dart</li>
<li>Firebase Authentication</li>
<li>Cloud Firestore</li>
<li>Hive (Local Storage)</li>
<li>BLoC State Management</li>
<li>Clean Architecture</li>
<li>SOLID Principles</li>
</ul>

<h2>Architecture</h2>

<h3>Presentation Layer</h3>
<ul>
<li>UI</li>
<li>BLoC (State Management)</li>
</ul>

<h3>Domain Layer</h3>
<ul>
<li>Entities</li>
<li>Use Cases</li>
<li>Repository Interfaces</li>
</ul>

<h3>Data Layer</h3>
<ul>
<li>Repository Implementations</li>
<li>Firebase Integration</li>
<li>Hive Local Data Source</li>
<li>Models</li>
</ul>

<p>
This structure ensures scalability, maintainability, and separation of concerns.
</p>
